# Розробка вебсайту для інтернет-магазину парфумерії "Marry Shine"

Кваліфікаційна робота на здобуття ступеня фахового молодшого бакалавра ([Закарпатський політехнічний фаховий коледж](https://zmt.com.ua/)).

Керівник роботи – **Леспух Олександр Михайлович**.

## Використані технології
* HTML
* CSS
* MySQL
* WordPress CMS
* Elementor
* WooCommerce

## Опублікований продукт
Розроблену вебплатформу для агрегації оголошень про продаж автомобілів опубліковано та розгорнуто у мережі Інтернет. Доступ до системи здійснюється за посиланням: ([Marry Shine](https://www.marryshine.shop/)).
